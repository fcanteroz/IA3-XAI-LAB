# IA3-XAI-LAB
This repository contains the source code of IA3 XAI LAB, an interactive web-based laboratory designed for data analysis, machine learning model training, and the application of explainability techniques to trained models.

IA3 XAI LAB provides an integrated environment that allows users to:
- Load and explore datasets in CSV and Excel formats.
- Perform exploratory data analysis (EDA).
- Train supervised learning models for both classification and regression tasks.
- Evaluate model performance using standard metrics and visualizations.
- Apply local and global explainability techniques to better understand model behavior.
- Analyze feature importance using different methodological approaches.
The tool has been developed as an accessible and interactive platform aimed at promoting transparency, interpretability, and trust in Artificial Intelligence systems. It is intended for both technical users and non-expert profiles who require an intuitive interface to experiment with machine learning models and their explanations within a unified environment.

The repository includes:
- The main application file (app.py).
- Modularized source code for classification, regression, plotting, and data handling.
- A requirements.txt file specifying all necessary dependencies.
- Supporting assets such as images and configuration files.
- Documentation and usage instructions.
